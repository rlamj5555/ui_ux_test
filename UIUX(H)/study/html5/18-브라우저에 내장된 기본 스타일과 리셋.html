Built-In CSS

어떤 이유로 브라우저가 "퍼블리셔가 작성한 CSS"를 읽지 못한다면,
브라우저 자체에 내장된 기본 스타일로 웹페이지에 적용하여 출력한다.


css reset(태그의 초기화)이 필요한 이유

웹 브라우저 마다 내장된 기본 스타일이 완전히 똑같지 않아서
크로스 브라우징을 구현하기 어렵기 때문에 자주 사용하는 태그들에 한해
기본 스타일 값을 제로화 한다.

약식초기화

css코딩에 들어가면서, 페이지에 속하는 모든 요소들의 여백(안쪽, 바깥쪽 모두)을 초기화시키는 작업.
*{margin:0;padding:0;}
